flex布局(兼容性)

发表于 2018-08-24 | 分类于 CSS

之前看了阮一峰讲解的flex布局，真的是觉得很好用啊，然后最近在做的一个项目中几乎都是以flex布局为主，然而，项目在测试阶段的时候，一个很严重的问题，一开始在做这个项目的时候，忘记考虑浏览器的兼容性了，泪崩~~然后我就仔仔细细的看了一下关于flex的兼容性，flex布局从旧版本dispaly: box;，到dispaly: flexbox;，再到现在的标准版本display: flex;。浏览器是从IE10以上开始支持flex布局的，但是IE10的是-ms形式的。下面是各个浏览器对flex布局的支持情况：

兼容性写法：

  1. display:box/flexbox/flex 的兼容
    display:-webkit-box;
    display: -moz-box;
    display: -ms-flexbox;
    display: -webkit-flex;
    display: flex;

  2. flex-direction的兼容
    -webkit-box-orient:vertical;
    -webkit-box-direction:normal;
    -moz-box-orient:vertical;
    -moz-box-direction:normal;
    flex-direction:column;
    -webkit-flex-direction:column;

  3. flex-wrap的兼容
    -webkit-flex-wrap:wrap;
    -webkit-box-lines:multiple;
    -moz-flex-wrap:wrap;
    flex-wrap:wrap;

  4. justify-content的兼容
    -webkit-justify-content:center;
    justify-content:center;
    -moz-box-pack:center;
    -webkit--moz-box-pack:center;
    box-pack:center;

  5. align-items的兼容
    align-items:center;
    -webkit-align-items:center;
    box-align:center;
    -moz-box-align:center;
    -webkit-box-align:center;
